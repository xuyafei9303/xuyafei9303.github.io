hexo配置github博客

##date: 2017-03-01 15:41:57

###tags:Github hexo

今天看到很多人完成了建立博客网站，所以按照hexo的方法建立了这个博客，具体地址是：
http://xuyafei9303.github.io/

我使用的是MacBook Pro，所以需要用到的是node.js和hexo。
node.js点这里https://nodejs.org/en/
下载对应的mac版本，会有一个pkg的安装器，点继续安装完成即可。

安装hexo。

sudo npm install -g hexo

期间需要输入电脑密码。
接着终端输入

hexo init

本地查看有没有成功：

hexo s

出现http：//localhost:4000/.Press Ctrl + C to Stop说明成功了。

github点进去注册：https://github.com/

账号注册完毕

github中需要配置ssh的key将本地和github建立一个远程连接，命令如下：

ssh-keygen

vim ~/.ssh/id_rsa.pub

将打开的口令复制下来《一大串的字符串》
：wq退出窗口。

来到github–>settings–>SSH and GPG keys–>添加key–>add

##开始上传博客
命令行依次执行：

npm install hexo-deployer-git –save

hexo g

hexo d

此时打开github会看到刚刚新建的仓库内有了内容，但是我们依然还没有开始写博客，接着：

hexo new “blogName”

打开hexo路径来到里面的source/_posts会看到你创建的blog，后缀是md的文件，用macdown等编辑器打开就可以按照macdown语法编写博客。

完成编辑以后，更新博客，此时编辑器必须是关闭的。

hexo g
hexo d

执行完刷新网页就能看到更新的内容了。
